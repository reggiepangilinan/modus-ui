# RadioButtonGroup

The `<RadioButtonGroup/>` component is used for selections that has one definitive answer.

Example Gender, Civil Status etc.

### Props

<Props of={RadioButtonGroup} />

## Default

```
const options: RadioButtonGroupOption[] = [
  { id: 'hot', text: 'This is Hot', value: 'hot' },
  { id: 'cold', text: 'This is Cold', value: 'cold' }
];

const [value, setvalue] = useState();

  <RadioButtonGroup
    label={props.label}
    disabled={props.disabled}
    options={options}
    value={value}
    onChange={(e: React.ChangeEvent<HTMLInputElement>): void => {
      setvalue(e.target.value);
    }}
  />
```

if you need a horizontal layout just set `itemsStyle={{ flexDirection: 'row' }}`
