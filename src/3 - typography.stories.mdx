## Font Family

**Font:** Nunito Sans

The primary font used by modus-ui is Nunito Sans. It then fallsback to the system fonts as an alternative.

```
font-family: 'Nunito Sans', -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto',
  'Oxygen', 'Ubuntu', 'Cantarell', 'Fira Sans', 'Droid Sans', 'Helvetica Neue',
  sans-serif !default;
```
